
# 集合QQ,网易,虾米,酷狗的音乐播放器.(主要为了练习vue语法)

# 运行

## 安装依赖包

```
$ npm install
```

## 运行web

web工程打包

```
$ npm run build
```

构建web工程

```
$ npm run dev
```
